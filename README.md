# qiime2-agent-skills
This repository contains skills for agentic AI assistants to work during QIIME 2 development. 

> [!WARNING]  
> **Experimental:** These skills are experimental. Please be aware that AI models, LLMs, and agents can make mistakes or hallucinate incorrect information. Always review the generated code and results carefully.

## Overview of Skills

* **`qiime2-plugin-creator`**: Scaffold new QIIME 2 plugins from the official template directly, without requiring external tools.
* **`qiime2-action-creator`**: Extend an existing QIIME 2 plugin by adding a CLI-backed action (new method or visualizer).
* **`qiime2-unit-test-writer`**: Write or update unit tests for QIIME 2 plugin code, following local repository test styles and mocking external boundaries.
